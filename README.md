# ui
# Get started 

This project is using [pnpm](https://pnpm.io/) as a package manager.

```

npm install -g pnpm
cd ui
pnpm install

```


### Build

To build all apps and packages, run the following command:

```

pnpm run build

```

### Develop

To develop all apps and packages, run the following command:

```

pnpm run dev

```
### Testing

To run jest

```

cd apps/docs
pnpm test 

```
