ECDPM - The Charter Project Africa Sample

Sample usage of the infographic inside a parent page

Simply insert a small script tag into the page at the position where you would like to include the infographic. Adjust the location of the script file to where you have uploaded the files.

How to use

Download the files, unzip the content and put them online at some location. Use the script tag below in your page. You need to configure the location of your files. You replace {location_of_your_files} with the domain and path where the files are located. By default the data (text) of the infographic is loaded from the data.json file. You can create your own file and load that instead by setting the location of the data_file in the configuration in this code.
